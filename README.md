# Inventory Optimiser App (Systems Development)

This repository contains a Google Colab prototype of an Inventory Optimiser App for:
- Retail chains (multi-store inventory visibility)
- Restaurants (ingredient-level stock and waste reduction)
- Distribution centres (bulk inventory movement)

## How to run
Open `InventoryOptimiserAPP.ipynb` in Google Colab and run cells top-to-bottom.

## Evidence included
- SQL + NoSQL persistence demonstration
- Secure authentication (hashing + login/logout)
- Custom REST API endpoints
- Cloud Function example code
- Unit tests (pytest)
